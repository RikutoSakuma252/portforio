マンツーマン指導で、3ヶ月で確実に体を変える。
力強くインパクトのあるデザインで、ジムのブランド力と信頼感を表現する。

- 20〜50代のビジネスパーソン
- ダイエット・ボディメイクに本気で取り組みたい人

## セクション構成

### 1. Hero
- フルスクリーン動画背景 or 強いインパクトの画像
- 大きなキャッチコピー: 「限界を、超えろ。」
- サブコピー: 「マンツーマン指導で、3ヶ月で体を変える。」
- CTAボタン（赤）: 「無料体験を申し込む」
- スクロール指示のアニメーション

### 2. Result（成果・実績）
- 数字で見せるセクション（カウントアップアニメーション）
  - 1,200+ 会員数
  - 98% 継続率
  - 3ヶ月 平均変化期間
  - 8年 運営実績

### 3. Program（プログラム）
- カード形式 3種
  - ボディメイク / 週2〜3回 / ¥49,800/月〜
  - ダイエット / 週2〜3回 / ¥49,800/月〜
  - アスリート強化 / 週3〜5回 / ¥79,800/月〜
- 各カード: プログラム名・ターゲット・内容・価格・CTAボタン
- 「おすすめ」バッジ付きカード

### 4. Why APEX（選ばれる理由）
- 左右交互レイアウト（画像 + テキスト）×3セット
  1. 完全マンツーマン指導 - トレーナーが徹底的に伴走
  2. 食事管理サポート - 専用アプリで毎日サポート
  3. 結果コミット制度 - 3ヶ月で結果が出なければ全額返金

### 5. Trainer（トレーナー紹介）
- 横スクロール or グリッドで3名紹介
  - 山田 健太 / 元プロアスリート / [placeholder]
  - 佐藤 純 / NSCA認定 / [placeholder]
  - 高橋 美穂 / 女性専門トレーナー / [placeholder]
- 写真・名前・資格・一言メッセージ

### 6. Voice（お客様の声）
- カード形式で3〜4件の体験談
  - アバター画像・名前（匿名でもOK）・年齢・職業・コメント
  - ビフォーアフターの数値（体重 -12kg など）
- 背景を少し明るめ（`#141414`）で区別

### 7. Flow（入会の流れ）
- 縦並び or 横並び4ステップ
  1. 無料カウンセリング予約（フォームまたはLINE）
  2. 無料体験トレーニング（60分）
  3. プログラム・料金のご説明
  4. ご入会・トレーニングスタート

### 8. Price（料金）
- シンプルな料金表（月額・回数券など）
- 初期費用・月額・その他費用を明記
- FAQを3〜5件追加（アコーディオン形式）

### 9. CTA（無料体験申し込み）
- 大きなCTAセクション（背景: アクセントカラー or 画像）
- 「まず、無料体験から。リスクは一切ありません。」
- フォーム（名前・電話番号・希望日時）or LINEボタン

### 10. Footer
- ロゴ・住所・電話 / SNSリンク・コピーライト

## React 設計方針

- `create-react-app` or Vite で構築
- コンポーネント分割: セクションごと + 再利用可能な小コンポーネント（`Card`, `Button`, `Badge` など）
- `useState` でFAQのアコーディオン管理
- カウントアップ: `useEffect` + `IntersectionObserver` で実装

## アニメーション方針

- Hero テキスト: Intersection Observer でのフェードイン
- Result 数字: カウントアップアニメーション（0 → 実数）
- Program カード: ホバーで赤いボーダー表示・背景色変化
- FAQ: `useState` でスムーズなアコーディオン開閉

# Site 05 - Flowly

## 基本情報

| 項目 | 内容 |
|------|------|
| サイト名 | Flowly（フローリー） |
| 業種 | 業務フロー管理SaaS LP |
| 技術スタック | Next.js + TypeScript + Tailwind CSS |
| 制作意図 | TypeScript・モダンSaaS LP のUI・複雑なアニメーションを示す |
| カラーコンセプト | インディゴ・ホワイト・ライトブルーグラデーション |

## コンセプト

> 「チームの「当たり前」を、もっとスマートに。」

中小企業向けの業務フロー管理ツール。
タスク・承認フロー・チャット・レポートを一つに集約し、
「Slack + Excel + メール」の煩雑さから解放する。
クリーンでテクノロジー感のあるデザインで、プロダクトの信頼性と使いやすさを訴求する。

## ターゲット

- 中小企業の経営者・マネージャー
- 社内の情報共有・タスク管理に課題を感じているチーム
- DXを進めたいが複雑なツールは避けたい企業

## Next.js + TypeScript 設計方針

- App Router + TypeScript strict mode
- 型定義: `types/` フォルダに集約（`Plan`, `Feature`, `Testimonial` など）
- コンポーネント: `components/ui/`（共通）+ `components/sections/`（セクション）
- アニメーション: Framer Motion
- 月払い/年払いトグル: `useState<'monthly' | 'yearly'>`
- FAQ アコーディオン: `useState` で開閉管理

## アニメーション方針

- Hero: テキストの段階的フェードイン（Framer Motion `staggerChildren`）
- Logos: CSS `animation: marquee` で無限スクロール
- Features: スクロール連動でモックアップ画像切り替え
- 数字: カウントアップアニメーション
- Pricing トグル: `AnimatePresence` で価格の切り替えアニメーション

## Unsplash / UI 画像方針

- Hero モックアップ: Figma or Canva でダッシュボード UI を自作し PNG 書き出し
- Testimonials: `https://i.pravatar.cc` でアバター画像生成
- Social Proof background: `analytics dashboard dark` (Unsplash)

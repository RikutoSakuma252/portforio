# Site 04 - HORIZON不動産

## 基本情報

| 項目 | 内容 |
|------|------|
| サイト名 | HORIZON不動産（ホライズン不動産） |
| 業種 | 不動産会社 コーポレートサイト（複数ページ構成） |
| 技術スタック | Next.js (App Router) + Tailwind CSS |
| 制作意図 | Next.js App Router・複数ページ・SSG/SSR の使い分けを示す |
| カラーコンセプト | ネイビー・ホワイト・ゴールドアクセント |

## コンセプト

> 「あなたの「理想」を、「現実」に。」

都市部・郊外の売買・賃貸を手がける地域密着型の不動産会社。
信頼・実績・地域への深い知識で、物件探しから引き渡しまで丁寧にサポート。
清潔感・信頼感・プロフェッショナルさを伝えるデザイン。

## ターゲット

- 住宅購入を検討している30〜50代
- 初めての不動産取引で不安を感じている人
- 地域に詳しい会社を探している人

## ページ構成（複数ページ）

```
/                    → トップページ（メイン LP 相当）
/properties          → 物件一覧（ダミーデータ）
/properties/[id]     → 物件詳細（動的ルート）
/company             → 会社概要
/contact             → お問い合わせ
```

## セクション構成（トップページ `/`）

### 1. Header（固定ナビゲーション）
- ロゴ + ナビリンク（物件一覧 / 会社概要 / お問い合わせ）
- 右端: 「物件を探す」ボタン（ゴールド）
- スクロールで背景色変化（透明 → ホワイト）

### 2. Hero
- 大きな背景画像（住宅・街並み）
- キャッチコピー: 「あなたの「理想」を、「現実」に。」
- サブコピー: 「地域に根ざして20年。あなたの大切な住まい探しを、丁寧にサポートします。」
- 物件検索ボックス（エリア・物件タイプ・価格帯のセレクト + 検索ボタン）

### 3. Feature（選ばれる3つの理由）
- アイコン + タイトル + テキストの3カラム
  1. 地域密着20年の実績
  2. 専任担当者によるワンストップサポート
  3. アフターフォローも安心

### 4. Properties（物件ピックアップ）
- 4〜6件の物件カード
  - 物件画像・物件名・所在地・価格・間取り・広さ
  - 「詳細を見る」ボタン → `/properties/[id]`
- 「物件一覧をすべて見る」ボタン → `/properties`

### 5. Flow（ご購入の流れ）
- 横並び or タイムライン形式で5ステップ
  1. ご相談・ヒアリング
  2. 物件のご紹介・内覧
  3. 売買契約の締結
  4. 住宅ローンのサポート
  5. 引き渡し・アフターフォロー

### 6. Voice（お客様の声）
- 3件の体験談カード
  - 名前（○○様）・年代・物件タイプ・コメント

### 7. Company（会社紹介ダイジェスト）
- 会社名・設立年・所在地・代表挨拶（一言）
- 「会社概要はこちら」→ `/company`

### 8. Contact CTA
- 「まずはお気軽にご相談ください」
- 電話番号 / LINE / お問い合わせフォームへのリンク

### 9. Footer
- ロゴ・会社情報・ナビリンク・コピーライト

## 物件一覧ページ（`/properties`）

- フィルター機能（エリア・価格・間取り）← `useState` で管理
- 物件カードグリッド（ダミーデータ 8〜12件）
- ページネーション（ダミー）

## 物件詳細ページ（`/properties/[id]`）

- 物件画像スライダー（4〜5枚）
- 物件名・価格・基本情報（間取り・広さ・築年数・所在地）
- 物件説明文
- お問い合わせフォーム（サイドバー）

## Next.js 設計方針

- App Router 採用
- トップページ・物件一覧: SSG（`generateStaticParams`）
- 物件詳細: 動的ルート（ダミーデータでSSG）
- メタデータ: `metadata` export でページごとに設定
- `next/image` で画像最適化

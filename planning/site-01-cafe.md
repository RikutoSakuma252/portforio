# Site 01 - KOKU Coffee

## 基本情報

| 項目 | 内容 |
|------|------|
| サイト名 | KOKU Coffee（コク コーヒー） |
| 業種 | スペシャルティコーヒーショップ LP |
| 技術スタック | HTML / CSS / JavaScript（バニラ） |
| 制作意図 | 素のHTML/CSS/JSでのレイアウト・アニメーション力を示す |

## コンセプト

> 「一杯に、物語を。」

スペシャルティコーヒーにこだわる小さなカフェ。
豆の産地・焙煎・抽出にこだわり、コーヒーを通じて日常に豊かさをもたらす。
シンプルで温かみのあるデザインで、ブランドの誠実さと品質感を表現する。

## ターゲット

- コーヒー好きな20〜40代
- 落ち着いた雰囲気のカフェを探している人
- 週末のひとときを特別にしたい人

## セクション構成

### 1. Hero
- フルスクリーン背景画像（コーヒーカップ・カフェ内観）
- キャッチコピー: 「一杯に、物語を。」
- サブコピー: 「こだわりの豆から、あなただけの一杯を。」
- CTAボタン: 「メニューを見る」→ #menu へスクロール

### 2. About
- 左: テキスト（カフェのこだわり・ストーリー）
- 右: カフェ外観または豆の画像
- テキスト例:
  - 「KOKU Coffeeは、2019年に東京・下北沢で誕生したスペシャルティコーヒーショップです。」
  - 「世界各地の農園から厳選した豆を、その豆の個性が最大限に引き出されるよう丁寧に焙煎しています。」

### 3. Menu
- カード形式で3〜4アイテム表示
  - エスプレッソ / ¥480
  - カフェラテ / ¥580
  - ハンドドリップコーヒー / ¥620
  - 本日のスペシャルティ / ¥780
- 各カードに画像・名前・価格・一言説明

### 4. Story（コーヒーへのこだわり）
- 横並び3カラム（アイコン + タイトル + テキスト）
  1. 産地へのこだわり - 農園直輸入のシングルオリジン
  2. 焙煎へのこだわり - 店内自家焙煎・少量ずつ丁寧に
  3. 抽出へのこだわり - バリスタが一杯ずつ丁寧に

### 5. Gallery
- 写真グリッド（3列 × 2行 = 6枚）
- カフェ内観・コーヒー・スイーツなど
- ホバーで軽いズームエフェクト

### 6. Access
- 店舗情報（住所・営業時間・定休日）
- Google Maps 埋め込み（ダミーでもOK）
- 住所: 東京都世田谷区下北沢X-X-X（[placeholder]）
- 営業時間: 10:00 - 20:00（月曜定休）

### 7. Footer
- ロゴ・コピーライト
- SNSアイコン（Instagram / Twitter）→ [placeholder]

## アニメーション方針

- スクロール連動: `IntersectionObserver` でフェードイン
- Hero テキスト: ページロード時にフェードアップ
- Menu カード: ホバーで軽い浮き上がり（`transform: translateY`）
- Gallery: ホバーでズーム + オーバーレイ
